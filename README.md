# PET FEAST FANTASY

An interactive virtual restaurant application where users serve absurdly themed meals to imaginary pets. This project uses a clean, multi-screen interface to simulate a service window experience.

## Features

* Character selection lobby for various imaginary pets
* Dual-character display featuring a guest pet and restaurant staff
* Randomized staff dialogue system
* Visual animations for pet reactions (dancing and jumping)
* Externalized logic and styling for better code organization
* Whimsical menu with unique meal options

## Technologies Used

* HTML
* CSS
* JavaScript

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/pet-feast-fantasy.git

```

### 2. Prepare Asset Files

Ensure the following image files are present in your project folder to match the code references:

* `unicorn.png`
* `dragon.png`
* `blob.png`
* `staff.png`

### 3. Run the Project

Simply open `index.html` in your web browser to start the application.
